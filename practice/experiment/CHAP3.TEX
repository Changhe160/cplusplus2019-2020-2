\section*{实验3.6\quad 计算求π的近似值}
\addcontentsline{toc}{section}{实验3.6\quad 计算求π的近似值}
编程计算π的近似值(要求小数点后面10位)。提示:可利用下列公式计算π值：
\par $\frac{\pi}{2}= 1+\frac{1}{3}+\frac{1}{3}\times\frac{2}{5}+\frac{1}{3}\times\frac{2}{5}\times\frac{3}{7}+\cdot \cdot \cdot \cdot \cdot \cdot \cdot $\
第n项为 $a_{n}$与前一项 $a_{n-1}$的关系是 $a_{n}=a_{n-1}\times\frac{n-1}{2n-1}$\\
【程序】\\
 \begin{lstlisting}[caption=*, label=prog3-7,basicstyle=\ttfamily,escapechar=`,frame=lines]
 #include <iostream>
using namespace std;
int main() {
    long int i(0);
    double sum(0), pi, term(1.);
    do {
        sum += term;
        i++;
        term = term*(i - 1) / (2 * i - 1.0);
    } while (term >= 1.0e-10);
    pi = 2 * sum;
    cout << "pi=" << pi << endl;
    return 0;
}
\end{lstlisting}

\section*{实验3.9\quad 对字符进行统计}
\addcontentsline{toc}{section}{实验3.9\quad 对字符进行统计}
输入若干字符，统计其中数字字符、空白字符和其他字符的个数，输入EOF结束。\\
【分析】
\par 要对字符进行统计，需要循环语句反复输入字符，读入字符后用switch 语句判断字符的种类：要统计3 种字符数，需要定义3个用于计数的变量 nDigit ,nWhite ,nOther,并置初始值0；读入字符用cin.get()函数，在这里用while循环比较合适。白字符指空白键、Tab键和回车键。EOF 表示End of File , 其值为-1，从键盘输入 CTRL+Z键即可。这里采用重载的 int cin.get( ) 函数，它返回的是整型数，所以能返回EOF。\\
【要求】
\begin{enumerate}
\item 程序中的变量c ，可以定义为char 类型吗？ 请试之，并解释原因。
\item 请注意程序中case 分支语句后的break语句，break能去掉吗？为什么？
\item 如果要分别统计0-9中个数字出现的次数，怎样才能有效地实现，请修改程序。
\item 如果要统计输入的一段文字中出现的行数、单词数和字符数，又怎样有效地实现，请完成之。
\end{enumerate}
【程序】%？？？？？？？？？？？？？？？？？？？？？？？？？？？？？？？
 \begin{lstlisting}[caption=*, label=prog3-9,basicstyle=\ttfamily,escapechar=`,frame=lines]
#include <iostream>
#include <stdlib.h>
using namespace std;
int main() {
  int c;
  int nWhite, nOther, nDigit;
  nWhite = nOther = nDigit = 0;
  c = cin.get();//`\color{green}从流中读取一个字符并返回一个整数`
  while (c != EOF) {
	 switch (c) {
	 case '0':case '1':case '2':case '3':case '4':
	 case '5':case '6':case '7':case '8':case '9':
		nDigit++;
		break;
	 case ' ':case '\n':case '\t':
		nWhite++;
		break;
	default:
		nOther++;
		break;
	}
	c=cin.get();//`\color{green}读入下一个字符`
  }
  cout << "Digits=" << '\t' << nDigit << '\n';
  cout << "White space=" << '\t' << nWhite << '\n';
  cout << "Other Chars=" << '\t' << nOther << '\n';
  system("pause");
  return 0;
}
\end{lstlisting}
